# Cloud KMS (Key Management Service)

KMS allows us to create and manage cryptographic keys, symmetric and asymmetric.

We can control their use in our applications and GCP Services.

KMS provides an API to encrypt, decrypt, or sign data. We can also use existing cryptographic keys created on premises.

KMS Integrates with almost all GCP services that need data encryption:

- **Google-managed key:** No configuration needed
- **Customer-managed key:** Use key from KMS
- **Customer-supplied key:** Provide your own key
