# Data states

There are types on data states depending if they are stored on device (HDD, DB, backups and archives) or being transferred across a network.

**Data at rest:** Stored on a device or a backup

**Data in motion:** Being transferred across a network

- Also called **data in transit**
- Examples:
  - Data copied from on-premise to cloud storage
  - An application talking to a database
- Two Types:
  - In and out of cloud (from internet)
  - Within cloud

**Data in use:** Active data processed in a non-persistent state (e.g. Data in RAM)
