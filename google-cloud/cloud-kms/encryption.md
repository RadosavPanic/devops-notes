# Encryption of data

Encryption is not sufficient only at rest, but as well in transit - between application to database.

**Symmetric Key Encryption:** This type algorithms use the same key for encryption and decryption.

## Asymmetric Key Encryption

It uses two type of keys: Public and Private. This is also called Public Key Cryptography.

We encrypt data with Public Key and decrypt with Private Key. Only Public Key should be shared.
