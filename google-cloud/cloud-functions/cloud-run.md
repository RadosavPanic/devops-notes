# Cloud Run & Cloud Run for Anthos

Cloud run is "**Container to Production in Seconds**". It's **fully managed** serverless platform for containerized applications.

Cloud run is built on top of an open standard - **Knative**.

Features:

- Zero infrastructure management
- Pay-per-use (For used CPU, Memory, Requests and Networking)

It provides fully integrated end-to-end developer experience:

- No limitations in languages, binaries and dependencies
- Easily portable because of container based architecture
- It integrates with: Cloud Code, Cloud Build, Cloud Monitoring & Cloud Logging Integrations
