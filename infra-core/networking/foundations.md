# Infra core networking

## NTP (Network Time Protocol)

NTP is used to sync PC clocks over a network to Coordinated Universal Time (UTC) within milliseconds.

    - It uses UDP port 123 and a hierarchical "stratum" system to ensure accuracy across devices.
    - Sync timestamps helps with log analysis, and ensures consistent certificates (SSL/TLS) and auth protocols.

0.ubuntu.pool.ntp.org - default Network Time Protocol NTP server address used by Ubuntu to sync a pc’s clock with accurate time servers over the internet.

## TDM (Time-Division Multiplexing)

TDM is a method to transmit multiple independent signals over a single, common communication channel.

## AMS (Application Management Services)

AMS are outsourced services for maintaining, supporting, and optimizing SW apps (like ERP, CRM or SaaS) to ensure high performance and security.
